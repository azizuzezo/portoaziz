# Profile photo

Currently in use: `aziz.png` (referenced from `lib/content/profile.ts` via `person.photoUrl`, `person.hasPhoto: true`).

To swap it, drop a new image in this folder and update `photoUrl`/`hasPhoto` in `lib/content/profile.ts` accordingly. Recommended: a square (1:1) image, at least 800x800px, face centered.
